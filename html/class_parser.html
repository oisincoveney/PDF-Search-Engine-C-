Detailed Description

The Parser class is a static class that contains the functionality for parsing a PDF and adding it to an IndexInterface index. The Parser class also keeps track of the total number of files, words, and pages parsed, as well as a table of documents and their respective word counts.

NOTE: The word counts are the counts of words that have been parsed, which does not include words shorter than 3, or stop words.
