Detailed Description

template<class T>
class Node< T >

The Node class are used as the main objects for the AVL tree. Nodes contain data and pointers to two different subnodes, known as children, and contain a height variable which contains the maximum height the tree.

This class will be used for the PDF Search Engine final project in SMU CSE 2341.

By: Oisin Coveney
